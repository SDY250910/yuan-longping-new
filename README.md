\# Prof. Yuan Longping Digital Archive



A complete digital archive commemorating Prof. Yuan Longping, the "Father of Hybrid Rice," featuring a responsive design, interactive timeline, and embedded Bilibili documentary.



\## Features



\- \*\*Modern Responsive Design\*\*: Fully responsive for all devices

\- \*\*Bilibili Video Integration\*\*: Embedded documentary from Bilibili

\- \*\*Interactive Timeline\*\*: Animated timeline of Prof. Yuan's life

\- \*\*Scientific Contributions Showcase\*\*: Interactive cards highlighting key achievements

\- \*\*Image Gallery\*\*: Visual legacy with three high-quality images

\- \*\*Smooth Animations\*\*: CSS and JavaScript animations

\- \*\*Accessible Navigation\*\*: Smooth scrolling and active link highlighting

\- \*\*No Large Files\*\*: No need to host large video files locally



\## Quick Start



1\. \*\*Download all files\*\* to a single folder

2\. \*\*Add your image\*\* to the `images` folder (name it `yuan\_field.jpg`)

3\. \*\*Open `index.html`\*\* in any modern browser

4\. The archive is ready to use!
